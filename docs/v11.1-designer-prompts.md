## §0. 每次开新 Figma 文件前的锚定提示词（必须先贴）

```text
你正在为 VibeHub v11.1 设计 UI。

产品一句话定位（12 个月不改）：
"VibeHub — 你和 AI 一起做的工作，有据可查。"

目标用户：中国独立开发者 / 一人公司（OPC）创始人。75% 非技术背景。
高频场景：每天在 /studio 里和 Agent 干活；每周导出 Ledger 给客户；每月更新公开信用名片。

视觉基因（"纸感工作台"）：
- 暖色护眼单主题，参照 Claude.ai / Claude Code 的纸质米白底
- 权重 50% Claude.ai · 20% Linear marketing 浅色 · 15% Stripe Docs ·
  10% Notion · 5% Apple HIG Light
- 不做暗色 / glassmorphism / neumorphism / glow / 政务红章 /
  Material Ripple / 漫画插画
- "丝滑" = 4 条全局缓动 + 锁死持续时间 + 几乎察觉不到的微动效
  不靠堆动画数量实现"丝滑"
- "清爽" = 每页只允许 1 个主色 CTA + 卡片每屏 5-7 张 +
  section 间距 48-96px + 段落行高 1.75

三件事架构（永不改）：
  Studio（做，每天）→ Ledger（签，每周）→ Card（晒，每月）

禁止项：
- 任何颜色硬编码 hex（必须使用 CSS 变量名，如 var(--color-primary)）
- 暗色 / 双主题切换器
- 大于 1 个主色 CTA 同屏
- 超过 6px 的 backdrop-blur
- 持续旋转的 loading spinner
- 红色小圆点未读提示（OPC 焦虑触发器）
- 动效超过 400ms 的进入过渡
- 任何情况下使用纯黑 #000 或纯白 #FFF（Hero 文字用暖深棕，背景用暖米白）

产出要求：
- 全部使用 Figma Variables 绑定颜色、圆角、间距
- 每个组件给出 default / hover / active / disabled / focus 五态
- 提供 desktop 1440 / tablet 1024 / mobile 375 三档响应式
- 所有文字按中文排版优化，中英夹杂时用非紧贴间距
- 所有数字、哈希、ID、时间戳必须使用 Geist Mono
- 所有 section 大标题（h1/h2）必须使用 Source Han Serif（思源宋体）

如有任何不确定，优先参考 Claude.ai 主界面而非 Linear。
```

### P1.4 全局动效规格（直接作为组件 interaction 配置）

```text
所有过渡必须从下表挑一项，不允许自定义：

行为                 时长 ms    缓动
hover 颜色           120       ease.standard
hover 阴影/transform 200       ease.decelerate
焦点环淡入           120       ease.standard
按钮按下             280       ease.spring，scale(0.98)
卡片 fade-up 进入    320       ease.decelerate，translateY(8 → 0)
Modal 打开           240       ease.decelerate，scale(0.96 → 1) + opacity
Modal 关闭           180       ease.accelerate
Drawer 进场          320       ease.standard
Dropdown 展开        180       ease.decelerate，scale(0.96 → 1)
Tab indicator 平移   280       ease.standard（motion.layoutId）
Tooltip 延迟         300ms 延迟后 150ms 淡入
Toast 进入/退出      240 / 180  decelerate / accelerate
Skeleton 循环        1400       ease.standard 无限循环
列表 stagger         30ms 间隔，每项 200ms decelerate
Accordion 展开       300       ease.decelerate
Accordion 收起       200       ease.accelerate

禁用的缓动：linear、ease-in-out 原生值。

微动效铁律：
1. 任何颜色变化必须 transition（不允许瞬切）
2. 可点击元素 hover 必须 translateY(-1px)（物理感）
3. 主按钮 active 必须 scale(0.98)
4. sticky 元素滚动 0-20px 阴影渐入 shadow.card → shadow.card.hover

a11y：
- @media (prefers-reduced-motion: reduce) → 所有动画 0.01ms
- 所有可交互元素 focus 必见 shadow.focus
```

## 附录 B · 每次设计 review 必过的 checklist

```text
交付一个页面前自检：

□ 页面只有 0-1 个 primary CTA
□ h1 / h2 使用 font.serif
□ 所有数字 / hash / 时间戳 使用 font.mono
□ 所有颜色绑定 Figma Variables（不允许在 layer 上写 hex）
□ 所有卡片有 shadow.card + hover shadow.card.hover
□ 所有 hover 有 translateY 和 color transition
□ 所有按钮有 focus-visible 态（shadow.focus）
□ 所有可点击元素尺寸 ≥ 44px（移动端）
□ 空态有 icon + title + description + CTA
□ 错误态有 color.error + 清晰文案
□ 加载态用 Skeleton 而非 spinner
□ 数据连接到 prisma model 的真实字段（附录中每个 page 有字段契约）
□ 响应式验证 375 / 1024 / 1440 三档
□ 没有使用禁用的 emoji / 红色圆点 / 政务红章 / glow
□ 文案避免 "协作网络" "工作台" "团队" 等 v8 词汇
```
