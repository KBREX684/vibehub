# VibeHub Roadmap (Current)

更新日期：2026-04-20

## 当前主线

VibeHub 当前唯一活跃主线是 **v11.0 最终章：AI 留痕本**。

唯一事实来源：
- `docs/v11.0-final-chapter-rfc.md`
- `docs/v11.0-backend-tasks.md`
- `docs/v11.0-frontend-tasks.md`

一句话定位：
> **VibeHub — 你和 AI 一起做的工作，有据可查。**

当前活跃产品面：
- `/`：公开首页
- `/studio`：个人工作站
- `/ledger`：操作账本
- `/u/[slug]`：公开 Trust Card
- `/pricing`
- `/settings/*`

当前活跃能力：
- Studio 中的个人工作记录、Agent 执行、artifact、快照与交付
- Ledger 签名链、导出、校验与锚定
- Trust Card 公开资料、指标与 PDF
- AIGC 标识、合规设置、审计轨迹
- 中国版计费与设置能力

## 当前约束

v11 之后默认冻结以下方向：
- Team Workspace 作为主产品重启
- Discover / 项目发现重启
- 协作撮合 / Matching / Collaboration Intent 重启
- 私信 / 聊天 / 内容流 / 排行榜 / 合集
- IDE / repo / CI / 自建 LLM / Agent 商店
- 海外支付与多语言主产品面

旧能力如果仍保留代码或 API，仅视为：
- 兼容路径
- 历史数据读取面
- 归档参考

不得再作为当前实现依据或产品叙事来源。
